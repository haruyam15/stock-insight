# 프로젝트 개요

## 프로젝트명

Stock Insight

## 프로젝트 목적

공공데이터(금융위원회 주식시세 API)를 기반으로 주식 초보자에게 친숙한 데이터 분석 서비스를 제공한다.
"어제 데이터를 기반으로 오늘을 공부하는 주식 분석 서비스"

실시간 매매 서비스 ❌
데이터 기반 주식 이해 서비스 ⭕

## 타겟 사용자

- 주식 입문자 (핵심 타겟)
- 데이터 기반 투자에 관심있는 일반인
- 금융 공부하는 학생

## 핵심 가치

1. 어려운 금융 데이터를 쉽게 설명
2. 수학 기반 분석 지표 제공 (이동평균, 표준편차, 상관계수 등)
3. 초보자 친화적 UI/UX

## 사용 데이터

| 항목 | 내용 |
|------|------|
| 데이터 출처 | 금융위원회 주식시세정보 API (공공데이터포털) |
| 갱신 주기 | T+1 (영업일 기준 다음날 오후 1시 이후) |
| 제공 항목 | 종목코드, 종목명, 시가, 종가, 고가, 저가, 거래량, 거래대금, 등락률 |
| MVP 범위 | 주식 시세(상장 주식)만 우선 구현 |

> 금요일 데이터는 차주 월요일(다음 영업일)에 제공된다.
> 예측 결과는 확률 형태로만 제공하며 투자 권유가 아닌 정보 제공 목적이다.

## MVP 핵심 기능

1. **종목 검색** - 종목명 또는 종목코드로 검색
2. **종목 상세 분석** - 가격 차트 + 수학 지표 + 초보자용 설명
3. **시장 리포트** - 오늘의 시장 요약 (상승/하락 종목 수, 주요 종목)
4. **TOP 랭킹** - 상승률 / 하락률 / 거래량 / 거래대금 TOP
5. **간단한 예측** - 과거 데이터 기반 확률형 예측

## 기술 스택

| 영역 | 기술 |
|------|------|
| 프론트엔드 | Next.js 14 (App Router), TypeScript, Tailwind CSS |
| 백엔드 | Next.js API Routes |
| 데이터베이스 | Supabase (PostgreSQL) |
| 데이터 수집 | Node.js 스크립트 (수동 실행 또는 cron) |
| 차트 | Recharts |

## 개발 순서

1. DB 스키마 (Supabase)
2. 데이터 수집 스크립트
3. API Routes
4. 분석 로직
5. 프론트엔드 페이지
