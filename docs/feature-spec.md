# 기능 명세서

## 1. 종목 검색

- 종목명 또는 종목코드로 검색
- 검색 결과 클릭 시 종목 상세 페이지로 이동
- 결과에 종목코드, 종목명, 전일 종가, 등락률 표시

## 2. 종목 상세 분석

### 기본 정보
- 종목명, 종목코드
- 시가 / 종가 / 고가 / 저가
- 거래량 / 거래대금
- 등락률 (전일 대비)

### 가격 차트
- 최근 30일 종가 라인 차트
- 이동평균선 오버레이 (MA5, MA20, MA60)

### 수학 지표
- 이동평균 (MA5, MA20, MA60)
- 변동성 (표준편차 기반)
- 최근 30일 상승 확률 (%)
- 추세 판단 (상승 추세 / 하락 추세 / 횡보)

### 초보자 설명
- 각 지표 옆에 쉬운 말로 설명 추가
- 예: "거래량이 평소보다 많다 = 이 종목에 관심이 몰리고 있다는 신호일 수 있어요"

### 비슷한 종목
- 최근 30일 종가 기준 상관계수가 높은 종목 3~5개 추천

## 3. 시장 리포트

매일 최신 기준일 데이터를 기반으로 자동 생성

- 전체 상승 종목 수 / 하락 종목 수
- 상승률 TOP 3 종목
- 하락률 TOP 3 종목
- 거래량 TOP 3 종목
- 시장 전체 평균 등락률

## 4. TOP 랭킹

탭으로 구분하여 제공

| 탭 | 내용 |
|----|------|
| 상승률 TOP | 당일 기준 상승률 높은 순 |
| 하락률 TOP | 당일 기준 하락률 높은 순 |
| 거래량 TOP | 당일 거래량 많은 순 |
| 거래대금 TOP | 당일 거래대금 많은 순 |

- 각 랭킹에서 클릭 시 종목 상세 페이지로 이동

## 5. 간단한 예측

> 투자 권유가 아닌 수학적 분석 결과를 정보 제공 목적으로만 제공

### 상승 확률
- 최근 30일 중 상승한 날의 비율
- 예: "최근 30일 기준 상승 확률 63%, 하락 확률 37%"

### 추세 신호
- MA5 > MA20 → 단기 상승 추세 가능성
- MA5 < MA20 → 단기 하락 추세 가능성
- MA20 > MA60 → 중기 상승 추세 가능성

### 변동성 등급
- 표준편차 기반으로 낮음 / 보통 / 높음 으로 표시
- 예: "이 종목은 변동성이 높은 편입니다. 가격이 크게 오르거나 내릴 수 있어요."

## 주식 초보 가이드 (각 지표 설명)

데이터와 연결된 교육 콘텐츠로 지표 옆에 노출

| 지표 | 초보자 설명 |
|------|------------|
| 거래량 | 오늘 이 주식이 몇 번이나 사고 팔렸는지 나타내요 |
| 이동평균선 | 일정 기간 동안의 평균 가격을 선으로 이은 것이에요 |
| 등락률 | 어제보다 오늘 가격이 얼마나 달라졌는지 비율이에요 |
| 변동성 | 가격이 얼마나 들쭉날쭉한지 나타내요 |
| 상승 확률 | 과거 데이터를 기반으로 계산한 확률이며 미래를 보장하지 않아요 |
