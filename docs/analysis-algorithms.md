# 분석 알고리즘

## 1. 이동평균선 (Moving Average)

일정 기간 동안의 종가 평균값. 가격 추세를 파악하는 데 사용한다.

```
MA(N) = 최근 N일 종가의 합 / N
```

| 구분 | 기간 | 의미 |
|------|------|------|
| MA5 | 5일 | 단기 추세 |
| MA20 | 20일 | 중단기 추세 (약 1개월) |
| MA60 | 60일 | 중장기 추세 (약 3개월) |

구현 위치: `src/lib/analysis/movingAverage.ts`

---

## 2. 변동성 (Volatility)

최근 N일 종가의 표준편차로 계산. 값이 클수록 가격 변동이 크다.

```
평균 = 최근 N일 종가의 합 / N
분산 = Σ(종가 - 평균)² / N
표준편차 = √분산
```

변동성 등급 기준 (MVP 기준, 추후 조정 가능):

| 등급 | 기준 |
|------|------|
| 낮음 | 표준편차 < 평균가의 2% |
| 보통 | 2% ~ 5% |
| 높음 | 5% 초과 |

구현 위치: `src/lib/analysis/volatility.ts`

---

## 3. 상승 확률 (Rise Probability)

최근 N일 중 전일 대비 종가가 오른 날의 비율.

```
상승 확률 = 상승한 날 수 / 전체 비교 가능한 날 수 × 100
```

- 기본 기간: 30일
- 상승 기준: 당일 종가 > 전일 종가

초보자 설명: "이 종목은 최근 30일 중 19일 올랐어요. 상승 확률 63%"

구현 위치: `src/lib/analysis/riseProbability.ts`

---

## 4. 추세 판단 (Trend Signal)

이동평균선 간의 관계로 추세를 판단한다.

| 조건 | 판단 |
|------|------|
| MA5 > MA20 AND MA20 > MA60 | 강한 상승 추세 |
| MA5 > MA20 | 단기 상승 추세 |
| MA5 < MA20 AND MA20 < MA60 | 강한 하락 추세 |
| MA5 < MA20 | 단기 하락 추세 |
| 그 외 | 횡보 |

구현 위치: `src/lib/analysis/trend.ts`

---

## 5. 상관계수 (Correlation Coefficient)

두 종목의 최근 N일 종가 변화율 사이의 피어슨 상관계수.
비슷한 종목 찾기에 사용한다.

```
r = Σ[(xi - x̄)(yi - ȳ)] / √[Σ(xi - x̄)² × Σ(yi - ȳ)²]
```

- 범위: -1 ~ 1
- 1에 가까울수록 같은 방향으로 움직이는 종목
- MVP에서는 사전 계산된 값을 DB에 저장하거나 상위 종목 일부만 계산

구현 위치: `src/lib/analysis/correlation.ts`

---

## 주의사항

- 모든 분석은 과거 데이터 기반이며 미래를 보장하지 않는다.
- 분석 결과 노출 시 반드시 "정보 제공 목적, 투자 권유 아님" 문구를 함께 표시한다.
- 데이터가 부족한 경우 (예: 상장된 지 얼마 안 된 종목) null 또는 "데이터 부족" 으로 처리한다.
